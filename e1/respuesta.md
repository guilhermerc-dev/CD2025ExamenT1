# Solucion Ejecicio 1 Code BLocks


Instalamos el code blocks con en comando en el terminal

> sudo apt-get install codeblocks

Ahora instalamos el compilador de c++ pero antes hacemos un **sudo update**

> sudo apt install build-essential

Abrimos el code blocks y abrimos el e1.cbp y ejecutamos el codigo

y ahora instalamos el paquete codeblocks-contrib


> sudo apt-get install codeblocks-contrib
